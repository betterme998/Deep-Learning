关于感知机,设定权重由人工进行的

神经网络的一个重要性质是它可以自动地从数据中学习到合适的权重参数

# 从感知机到神经网络

1. 神经网络的例子  
   n层神经元  
   最左边的一列称为输入层(第0层)，最右边的一列称为输出层(地n-1层)，中间的一列称为中间层(隐藏层)。

2. 神经网络中信号是如何传递的呢？  
   复习感知机：y = h(b + w1x1 + w2x2) # 输入信号的总和会被函数h(x)转换

b是被称为偏置的参数，用于控制神经元被激活的容易程度；  
而w1和w2是表示各个信号的权重的参数，用于控制各个信号的重要性。

3. 激活函数登场  
   h（x）函数会将输入信号的总和转换为输出信号，这种函数一般称为激活函数（activation function）。  
   激活函数的作用在于决定如何来激活输入信号的总和

a = b + w1x1 + w2x2 # 计算输入信号的加权总和  
y = h(a) #激活函数转换这一总和  
激活函数是连接感知机和神经网络的桥梁。A

3.1.阶跃函数:  
一旦输入超过阈值，就切换输出。 h(x) = 0 (x <= 0) or 1 (x > 0)  
激活函数从阶跃函数换成其他函数，就可以进入神经网络的世界了

3.2.神经网络使用的激活函数  
sigmoid函数：  
h(x) = 1 / (1 + exp(-x)) # exp(−x)表示e−x(e的负x的平方)的意思。e是纳皮尔常数2.7182 . . .。
