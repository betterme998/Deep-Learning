# 从感知机到神经网络

1. 神经网络的例子  
   n层神经元  
   最左边的一列称为输入层(第0层)，最右边的一列称为输出层(地n-1层)，中间的一列称为中间层(隐藏层)。

2. 神经网络中信号是如何传递的呢？  
   复习感知机：y = h(b + w1x1 + w2x2) # 输入信号的总和会被函数h(x)转换

b是被称为偏置的参数，用于控制神经元被激活的容易程度；  
而w1和w2是表示各个信号的权重的参数，用于控制各个信号的重要性。

3. 激活函数登场  
   h（x）函数会将输入信号的总和转换为输出信号，这种函数一般称为激活函数（activation function）。  
   激活函数的作用在于决定如何来激活输入信号的总和

a = b + w1x1 + w2x2 # 计算输入信号的加权总和  
y = h(a) #激活函数转换这一总和  
激活函数是连接感知机和神经网络的桥梁。A

3.1.阶跃函数:  
一旦输入超过阈值，就切换输出。 h(x) = 0 (x <= 0) or 1 (x > 0)  
激活函数从阶跃函数换成其他函数，就可以进入神经网络的世界了

3.2.神经网络使用的激活函数  
sigmoid函数：  
h(x) = 1 / (1 + exp(-x)) # exp(−x)表示e−x(e的负x的平方)的意思。e是纳皮尔常数2.7182 . . .。

# sigmoid函数和阶跃函数的比较

不同点：
.sigmoid函数是一条平滑的曲线，输出随着输入发生连续性的变化。  
.阶跃函数以0为界，输出发生急剧性的变化

也就是说，感知机中神经元之间流动的是0或1的二元信号，  
而神经网络中流动的是连续的实数值信号。 （0.731 . . .、0.880 . . .等实数）

sigmoid函数的平滑性对神经网络的学习具有重要意义

共同点：  
当输入信号为重要信息时，阶跃函数和sigmoid函数都会输出较大的值；  
当输入信号为不重要的信息时，两者都输出较小的值。  
还有一个共同点是，不管输入信号有多小，或者有多大，输出信号的值都在0到1之间。

# 非线性函数

阶跃函数和sigmoid函数均为非线性函数  
sigmoid函数是一条曲线，阶跃函数是一条像阶梯一样的折线

线性函数：输出值 是输入值 的常数倍的函数称为线性函数 h(x) = cx。c为常数  
线性函数是一条笔直的直线

神经网络的激活函数必须使用非线性函数  
为什么？  
如果使用线性函数：h(x) = cx 作为激活函数  
把y(x) = h(h(h(x)))的运算对应3层神经网络 也就是：y(x) = c × c × c × x  
但是同样处理可以由y(x) = ax（注意，a = c × c × c）一次运算完成  
使用线性函数时，无法发挥多层网络带来的优势。  
激活函数必须使用非线性函数。

# 多维数组的运算

多维数组: 数字的集合  
数字排成一列的集合、排成长方形的集合、排成三维状或者（更加一般化的）N维状的集合都称为多维数组。

数组的维数：通过np.dim()函数获得。  
数组的形状：通过实例变量shape获得  
A.shape的结果是个元组（tuple）

np.array([[1,2], [3,4], [5,6]])的形状是(3, 2)  
[[1 2]  
 [3 4]  
 [5 6]]  
3 × 2的数组表示第一个维度有3个元素，第二个维度有2个元素。  
第一个维度对应第0维，第二个维度对应第1维（Python的索引从0开始）  
二维数组也称为矩阵（matrix）  
数组的横向排列称为行（row），纵向排列称为列（column）。

.矩阵的乘积的计算方法  
左边矩阵的行（横向）和右边矩阵的列（纵向）以对应元素的方式相乘后再求和而得到的  
运算的结果保存为新的多维数组的元素

矩阵的形状  
(3, 2) (2, 3) 或者 (2, 3) (3, 2)  
矩阵A和矩阵B的对应维度的元素个数必须保持一致。

(3, 2) (2, 3) # B\*A的维度(3, 3)  
矩阵乘积的形状由，矩阵A的第1维和矩阵C的第0维的元素个数

(3, 2) (2,)  
A是二维矩阵、B是一维数组时，也要保持对应维度的元素个数一致

# 3层神经网络的实现

3层神经网络：  
 输入层（第0层）有2个神经元x1,x2  
 第1个隐藏层（第1层）有3个神经元a1,a2,a3  
 第2个隐藏层（第2层）有2个神经元a1,a2 和上面不同  
 输出层（第3层）有2个神经元y1, y2

偏置：b1  
权重：W11, W12 (W11：地一层的第一个神经元的权重，W12：地一层的第二个神经元的权重)  
输入信号：x1, x2

第一层：  
a1 = W11x1 + W12x2 + b1

第一层：

# 输出层的设计

神经网络可以用在分类问题和回归问题上，不过需要根据情况改变输出层的激活函数  
回归问题：恒等函数  
二元分类：sigmoid函数  
多元分类：softmax函数

分类问题：数据属于哪一个类别的问题 （区分图像中的人是男性还是女性的问题就是分类问题）  
回归问题：根据某个输入预测一个（连续的）数值的问题（根据一个人的图像预测这个人的体重的问题就是回归问题）

.1.恒等函数和softmax函数

恒等函数：将输入按原样输出
