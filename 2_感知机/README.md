感知机是由美国学者Frank Rosenblatt在1957年提出来的。  
为何我们现在还要学习这一很久以前就有的算法呢？  
因为感知机也是作为神经网络（深度学习）的起源的算法。  
因此，学习感知机的构造也就是学习通向神经网络和深度学习的一种重要思想。

# 感知机是什么

感知机接受多个输入信号，通过加权求和再经过一个符号函数，输出一个信号（+1或0）来判断类别

数学公式表示：  
y = (w1*x1 + w2*x2 <= 偏置) or y = (w1*x1 + w2*x2 > 偏置)
x1,x2:输入值  
w1,w2:权重  
偏置：  
y:输出信号

.神经元：是构成神经网络的基本计算单元  
简单理解：神经元接收多个输入，对它们加权求和，再加上一个偏置，最后通过一个激活函数得到输出

.输入：来自前一层或其他数据源

.权重与偏置：可学习的参数，决定每个输入的重要性以及激活的难易程度

.激活函数：引入非线性（如Sigmoid，ReLU）感知机用的符号函数也是激活函数的一种

.输出：传递到下一层神经元

输入信号被送往神经元时 ,会被分别乘以固定的权重(w1x1,w2x2).神经元会计算传过来的信号总和,只有当这个总和超过了某个界限时,才会输出1  
也被称为"神经元激活"  
这里将界限称为"阈值"

# 简单逻辑电路

1.与门  
与门是有两个输入和一个输出的门电路,与门仅在两个输入均为1时输出1,其他时候输出0  
x1,x2,y  
0,0,0  
1,0,0  
0,1,0  
1,1,1

感知机如何实现与门?  
当(w1,w2,偏置) = (0.5,0.5,0.7)时,(0.5,0.5,0.8)时,或者(1.0,1.0,1.0)时,仅当x1和x2均为1时,信号的加权总和才会超过给定的阈值

2.与非门和或门  
与非门：颠倒了与门逻辑的输出  
x1,x2,y  
0,0,1  
1,0,1  
0,1,1  
1,1,0

感知机如何实现与非门?  
当(w1,w2,偏置) = (-0.5,-0.5,-0.7)时（无限个）

或门：只要有一个输入信号是1，输出就为1  
0,0,0
1,0,1  
0,1,1  
1,1,1

注意：这里决定感知机参数的并不是计算机，而是我们人。  
而机器学习的课题就是将这个决定参数值的工作交由计算机自动进行。  
学习是确定合适参数的过程。
而人要做的是思考感知机的构造（模型），并把训练数据交给计算机。

与门，与非门，或门的感知机构造是一样的。  
实际上，3个门电路只有参数的（权重和阈值）不同。也就是说通过适当调整参数值，就可以变为与门，与非门，或门。
